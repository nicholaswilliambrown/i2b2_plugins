Overview:
This plugin will create a new patient set based on the two sets and selected relationship type. The new patient set will contain the subset of patient set 1 who have the selected relationship to a member of set 2.

Instructions:
  • Drop a Patient set into the Patient Set 1 input box. This box should countain the source of the relationhip, i.e. the mothers in a MotherOf query.
  • Drop a Patient set into the Patient Set 2 input box. This box should countain the target of the relationhip, i.e. the children in a MotherOf query.
  • Select the relationship type from the dropdown.
  • Enter a name for the query into the Query Name box. This will be the query name for this patient set in your previous queries window
  • Click the "View Results" tab to generate the results.
  • Wait for the View Results tab to populate. This will tell you the number of patients in the resulting set.
  • Refresh your Previous Queries window to view patient set, or use it in further queries.
